# 🦠 Аналіз та Прогнозування COVID-19 (Data Science & ML Project)

Цей проєкт являє собою повноцінний пайплайн (End-to-End Pipeline) для роботи з даними: від завантаження та очищення "сирого" датасету про COVID-19 до побудови складних візуалізацій та тренування моделей машинного навчання.

## 🚀 Головні фічі проєкту (Що робить скрипт)

1. **🧹 Повний ETL-процес (Очищення та Підготовка):**
   - Обробка пропущених значень та дублікатів.
   - Трансформація даних та генерація нових ознак (наприклад, `growth_rate_new_cases`).
   - Кодування категоріальних змінних (One-Hot Encoding, LabelEncoder) та масштабування (StandardScaler).

2. **📊 Автоматична Візуалізація:**
   - На відміну від стандартних скриптів, код **працює у фоновому режимі** і не зупиняється для показу графіків (`plt.show()`). 
   - Замість цього скрипт автоматично генерує та зберігає **10 PNG-зображень** високої якості (тренди, гістограми, теплові карти кореляції, матриці плутанини тощо).

3. **🤖 Machine Learning (Регресія та Класифікація):**
   - **Регресія** (прогнозування `new_cases`): Linear, Polynomial (degree=2), Ridge, Lasso.
   - **Класифікація** (виявлення спалахів `new_cases > 1000`): Logistic Regression, Decision Tree, Random Forest, k-NN, Gradient Boosting.
   - **SMOTE:** Балансування класів для усунення дисбалансу та мінімізації пропущених спалахів (False Negatives).

4. **⚙️ Оптимізація та Аналіз:**
   - **SelectKBest:** Зменшення розмірності даних до Топ-10 найважливіших ознак.
   - **GridSearchCV:** Автоматичний підбір найкращих гіперпараметрів для моделі Random Forest.
   - **Cross-Validation:** Перевірка стабільності моделей (5-Fold CV).

## 🛠 Встановлення та Запуск

**Вимоги:**
- Python 3.10+
- Встановлені залежності з файлу `requirements.txt`

**Крок 1. Встановлення бібліотек**
Відкрийте термінал у папці з проєктом та виконайте:
```bash
pip install -r requirements.txt
```
**Крок 2. Запуск проєкту**

```bash
python main.py
```
**Скрипт виконається за 1-3 хвилини і самостійно створить усі очищені датасети та графіки у вашій папці.**

**📁 Структура проєкту**
**Після запуску коду ваша папка виглядатиме так:**

```bash
├── main.py                     # Головний скрипт з усім пайплайном
├── requirements.txt            # Список залежностей
├── covid_data.csv              # Вхідний (оригінальний) датасет
├── covid_clean.csv             # Очищені дані (генерується скриптом)
```
